# 📊 MyFinance - Gestor Financeiro Inteligente

**MyFinance** é uma aplicação web robusta e moderna para gestão de finanças pessoais. Desenvolvida para ser 100% focada em privacidade, ela funciona inteiramente no lado do cliente, garantindo que seus dados nunca saiam do seu navegador.

## 🌟 Funcionalidades Principais

- **🎯 Dashboard Interativo**: Visualize seus indicadores de saúde financeira (Total Cartão, Manual, Renda e Saldo Final) em tempo real.
- **📈 Histórico Anual Estruturado**: Gráfico comparativo de Gasto vs. Renda de Janeiro a Dezembro para análise de evolução.
- **📂 Importação Inteligente de CSV**: Motor de categorização automática baseado em palavras-chave para faturas de cartão de crédito.
- **🗓️ Parcelamento Automático**: Função exclusiva para replicar gastos parcelados para meses futuros com correção inteligente de datas (erro do dia 31).
- **🗳️ Backup e Restauração**: Exporte e importe seus dados em formato JSON para garantir a persistência das informações.
- **📊 Navegação por Cliques**: Clique em qualquer barra do gráfico anual para viajar instantaneamente para os detalhes daquele mês.
- **🌗 Dark Mode Premium**: Interface moderna com design focado em usabilidade e estética (Glassmorphism).

## 🛠️ Tecnologias Utilizadas

- **Estrutura**: HTML5 Semântico
- **Estilização**: CSS3 Moderno (Custom Properties, Flexbox, Grid)
- **Lógica**: JavaScript Vanilla (ES6+)
- **Bibliotecas**:
  - [Chart.js](https://www.chartjs.org/) para visualizações de dados.
  - [Lucide Icons](https://lucide.dev/) para iconografia moderna.
  - [PapaParse](https://www.papaparse.com/) para processamento de arquivos CSV.
  - LocalStorage API para persistência de dados local.

## 🚀 Como Executar o Projeto

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. **Dica**: Para uma melhor experiência de carregamento de ícones e bibliotecas, recomenda-se abrir usando um servidor local (como a extensão "Live Server" do VS Code ou `python -m http.server`).
